# spa-test-react-ng

## How to start

### angular-app

- cd angular-app
- npm install
- yarn
- npm run serve:single-spa:angular-app

### react-app

- cd react-app
- npm install
- yarn
- npm start

### root-config

- cd root-config
- npm install
- yarn
- npm start

- open browser http://localhost:9000/
